关于 nan 和 None

dataframe中默认的空值为NaN，这是numpy中的一个类型，特殊的 float。
Series和DataFrame都会自动把 None 转换成 NaN 然后 运算的时候会把 NaN 当成 0。

详细可以参考：pandas numpy 处理缺失值，none 与 nan 比较
