Query hive data

иғҢжҷҜ

жңүHueеҸҜд»Ҙз”ЁжқҘжҹҘиҜўhiveе’Ңimpalaзҡ„ж•°жҚ®пјҢдҪҶжҳҜдҪҝз”Ёиө·жқҘдёҚжҳҜзү№еҲ«ж–№дҫҝпјҢе°Өе…¶жғіиҰҒеҗҢж—¶жҠҠж•°жҚ®еҜјеҮәжқҘзҡ„ж—¶еҖҷгҖӮ

еҺҹжқҘжғіе°қиҜ•з”Ёjavaзҡ„ж–№ејҸпјҢд№ҹеҗ‘еҗҢдәӢиҰҒдәҶdemoпјҢдҪҶжҳҜйңҖиҰҒkeytabпјҢиҖҢдё”иҝҳжҳҜеҖҫеҗ‘дәҺз”ЁpythonгҖӮ

жңҖеҗҺе°қиҜ•дәҶдёҖдәӣpythonеә“еӨұиҙҘ(дё»иҰҒжҳҜиҝҷдәӣmoduleдҫқиө–зҡ„дёңиҘҝжҜ”иҫғеӨҡпјҢйңҖиҰҒеҫҲеӨҡзҺҜеўғзҡ„й…ҚзҪ®пјҢдҪҶжҳҜйғҪжІЎжңүжқғйҷҗпјҢжҜ”иҫғйә»зғҰ)пјҢд»ҺеҗҢдәӢйӮЈеҫ—зҹҘд»–д»¬зҡ„Tableauжң¬ең°з”Ёзҡ„жҳҜodbcзҡ„иҝһжҺҘж–№ејҸгҖӮ

д№ӢеүҚиҜ•иҝҮз”ЁpyodbcеҺ»иҝһжҺҘsql serverж•°жҚ®еә“пјҢиҖҢдё”зҗҶи®әдёҠдҪҝз”ЁodbcпјҢеҸӘиҰҒж•°жҚ®жәҗй…ҚзҪ®еҘҪе°ұеҸҜд»ҘдәҶпјҢapiзҡ„дҪҝз”ЁжҳҜе…ұйҖҡзҡ„пјҢдёҚз®Ўд»Җд№ҲиҜӯиЁҖйғҪеҸҜд»Ҙз”Ёodbcзҡ„ж–№ејҸеҺ»и®ҝй—®гҖӮ

з¬¬дёүж–№иҪҜд»¶е®үиЈ…

  1. Cloudera hive odbc (з”ЁдҪҝз”Ёimpalaзҡ„иҜқе°ұе®үиЈ… impala odbc)пјҢ е№¶й…ҚзҪ®еҘҪиҝһжҺҘдёҺж•°жҚ®жәҗ
  2. MIT Kerberos (йңҖиҰҒдҪҝз”ЁKerberosи®ӨиҜҒ)

дё»з•Ңйқў

yamlжҳҜз”ЁжқҘй…ҚзҪ®еҹәжң¬дҝЎжҒҜпјҢеғҸжңҖеҗҺж–Үд»¶иҫ“еҮәзҡ„зӣ®еҪ•д№ҹеҸҜд»Ҙй…ҚзҪ®пјҢеҪ“еүҚй»ҳи®ӨoutputпјҢlibж”ҫдәҶдёҖдәӣиҮӘе·ұе°ҒиЈ…зҡ„зұ»е’Ңж–№жі•.

дё»з•ҢйқўеҸҜд»ҘзңӢеҲ°еҸҜд»ҘйҖүжӢ©дёҚеҗҢзҡ„ж•°жҚ®жәҗпјҲеңЁй…ҚзҪ®ж–Үд»¶йҮҢй…ҚзҪ®еҲқе§Ӣзҡ„ж•°жҚ®пјүпјҢиҫ“е…ҘжңҖеҗҺиҰҒеӯҳзҡ„ж–Үд»¶еҗҚеӯ—пјҲй»ҳи®ӨжҳҜtest.{current_datetime}.csv)пјҢиҫ“еҮәзҡ„ж–Үд»¶зұ»еһӢ(csv/excel)пјҢд»ҘеҸҠдҪ зҡ„sqlиҜӯеҸҘгҖӮ

дёәдәҶжӣҙзҒөжҙ»пјҢеҪ“startд№ӢеүҚпјҢдҪ еҸҜд»Ҙжӣҙж–°дҝЎжҒҜпјҢйҮҚж–°queryпјҢеҚіж”ҜжҢҒеӨҡзәҝзЁӢжҹҘиҜўпјҢдёҚиҝҮдёҚе»әи®®дёҖж¬ЎжҹҘеӨӘеӨҡпјҢиҖҢдё”иҰҒжіЁж„Ҹиҫ“еҮәж–Үд»¶зҡ„еҗҚеӯ—пјҢеҰӮжһңйҮҚеҗҚзҡ„иҜқж•°жҚ®е°ұдјҡиў«иҰҶзӣ–гҖӮ

дё»иҰҒжҹҘиҜўзҡ„д»Јз ҒеҰӮдёӢеӣҫжүҖзӨәпјҢе°ҒиЈ…дәҶHiveUtilsеҒҡдёҖдәӣз»ҶиҠӮеӨ„зҗҶгҖӮ

дёҠйқўзҡ„е·Ҙе…·ж»Ўи¶ідәҶзү№ж®Ҡзҡ„sqlзҡ„йңҖиҰҒпјҢдҪҶжҳҜжңүж—¶еҖҷпјҢжҲ‘д»¬еёҢжңӣдёҖж¬ЎиғҪжҠҠи®ёеӨҡtable/viewзҡ„ж•°жҚ®еҜјеҮәжқҘжҹҘзңӢгҖӮ

жүҖд»Ҙд№ҹеҶҷдәҶеҸҰеӨ–зҡ„и„ҡжң¬пјҢйҖҡиҝҮжҠҠжғіиҰҒеҜјеҮәжқҘзҡ„table/viewеҗҚеӯ—еҲ—еңЁй…ҚзҪ®ж–Үд»¶йҮҢпјҢдёҖж¬ЎжҖ§иө·еӨҡдёӘзәҝзЁӢеҺ»жҹҘиҜўгҖӮ

VMware Centos7 Network Setup

иғҢжҷҜ

жІЎжғіеҲ°жҗһиҷҡжӢҹжңәжҗһдәҶиҝҷд№Ҳд№…пјҢз»ҲдәҺй…ҚзҪ®жҲҗеҠҹдәҶпјҢдёҚиҝҮиё©еқ‘д№ҹжҳҜйҡҫе…Қзҡ„пјҢи®°еҪ•дёҖдёӢгҖӮ

зӣ®еүҚиҰҒеҒҡзҡ„жҳҜжҗӯе»ә Hadoop йӣҶзҫӨзҺҜеўғпјҢдҪҶжҳҜеңЁй…ҚзҪ®еӨҡеҸ° linux иҷҡжӢҹжңәж—¶зҪ‘з»ңдёҖзӣҙжІЎжңүеј„еҘҪпјҢиҝңзЁӢж— жі•и®ҝй—®пјҢеҮәзҺ°еҗ„з§Қй—®йўҳгҖӮдё»иҰҒеҺҹеӣ жҳҜиҮӘе·ұиҝҷеқ—еҹәзЎҖдёҚзүўпјҢзҪ‘дёҠзҡ„иө„ж–ҷд№ҹеӨӘжқӮгҖӮ

жӣҙйҮҚиҰҒзҡ„жҳҜпјҢжҲ‘дҪҝз”Ёзҡ„жҳҜеёҰ GUI зҡ„ CentOSпјҢеҜјиҮҙдәҶжңүдёӨеҘ— Network з®ЎзҗҶпјҢжүҖд»ҘеҶІзӘҒдәҶпјҢе°ҙе°¬гҖӮ

зҺҜеўғ

  • VMWare 12
  • CentOS7
  • дё»жңә Win10
  • NAT жЁЎејҸ

NAT жЁЎејҸ

еӣ дёәйҮҮз”Ё NAT жЁЎејҸеҜ№иҷҡжӢҹжңәиҝӣиЎҢзҪ‘з»ңй…ҚзҪ®пјҢжүҖд»ҘйңҖиҰҒе°Ҷдё»жңәзҪ‘з»ңе…ұдә«з»ҷиҷҡжӢҹжңә NAT зҪ‘еҚЎпјҢеҰӮдёӢпјҡ

еҗҜз”Ёд№ӢеҗҺ, еҲҷ VMnet8 зҪ‘еҚЎзҡ„й…ҚзҪ®иҰҒеҰӮдёӢжүҖзӨәпјҢеӣәе®ҡ ip зҪ‘ж®өдёә 192.168.137.*

CentOS7 зҪ‘з»ңй…ҚзҪ®

дёӢйқўд»Ҙе…ӢйҡҶзҡ„дёҖеҸ°жңәеӯҗдёәдҫӢпјҲиў«е…ӢйҡҶзҡ„жңәеӯҗдёә hadoop00/192.168.137.100пјүпјҢй…ҚзҪ®ж–°зҡ„зҪ‘з»ңгҖӮ

Mac Address

еҪ“еҲҡе…ӢйҡҶзҡ„ж—¶еҖҷпјҢжңәеӯҗзҡ„ Mac ең°еқҖд№ҹжҳҜдёҖж ·зҡ„пјҢжүҖд»ҘиҰҒеңЁеҗҜеҠЁеүҚе…ҲйҮҚж–°з”ҹжҲҗпјҲ00:50:56:24:A9:10 е°ұжҳҜж–°з”ҹжҲҗзҡ„ mac ең°еқҖпјүпјҡ

hostname

еҺҹжқҘжңәеҷЁзҡ„ hostname жҳҜ hadoop00пјҢжүҖд»ҘжҲ‘д»¬йңҖиҰҒдҪҝз”Ёhostnamectl set-hostname hadoop01е°ҶеҗҚеӯ—ж”№жҲҗhadoop01

/etc/hosts

дёӢйқўжҳҜеўһеҠ  hosts й…ҚзҪ®пјҢеңЁд№ӢеҗҺе»әз«ӢйӣҶзҫӨж—¶пјҢйңҖиҰҒе°ҶеӨҡеҸ°жңәеӯҗзҡ„ ipпјҢhostname дҝЎжҒҜйғҪиҰҒж·»еҠ иҝӣжқҘгҖӮ

дҝ®ж”№зҪ‘еҚЎй…ҚзҪ®

йҰ–е…ҲжҲ‘д»¬еҸҜд»ҘдҪҝз”Ёifconfigе…ҲжҹҘзңӢдёҖдёӢзҪ‘еҚЎдҝЎжҒҜпјҲиҝҷжҳҜжҲ‘е·Із»Ҹй…ҚзҪ®еҘҪдәҶпјүпјҢ
жҲ‘д»¬еҸҜд»ҘзңӢеҲ°зҪ‘еҚЎзҡ„еҗҚеӯ—дёәeno16777736, Mac ең°еқҖе’ҢжҲ‘д»¬з”ҹжҲҗзҡ„жҳҜдёҖиҮҙзҡ„гҖӮ

жүҖд»ҘжҲ‘д»¬йңҖиҰҒй…ҚзҪ®/etc/sysconfig/network-scripts/ifcfg-eno16777736ж–Үд»¶пјҢиҝҷйҮҢдҫҝжҳҜж ёеҝғзҡ„й…ҚзҪ®гҖӮ

еҶ…е®№еҰӮдёӢжүҖзӨәпјҡ

д№ӢеҗҺжңүдёҖдәӣиө„ж–ҷдёӯиҜҙиҰҒеҲ йҷӨ/etc/udev/rules.d/70-persistent-ipoib.rules,дҪҶжҲ‘жҹҘзңӢиҝҮпјҢйҮҢйқўжҳҜз©әзҡ„пјҢеҸҜиғҪжҳҜжҲ‘дҪҝз”Ёзҡ„зі»з»ҹе…ізі»пјҢжғіжқҘдёҚеҲ йҷӨд№ҹжІЎжңүе…ізі»зҡ„гҖӮ

йҮҚеҗҜзҪ‘еҚЎжңҚеҠЎ

жӯӨеӨ–пјҢдёәдәҶж–№дҫҝпјҢйҳІзҒ«еўҷеҸҜд»Ҙе…ій—ӯжҺүпјҡ
