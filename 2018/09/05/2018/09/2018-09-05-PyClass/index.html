那他们的区别是什么呢？

static method

静态方法其实就是一个单纯的方法，相当于把类外的一个普通方法挪到这个类中，通过类名访问。但是他其实是和这个类及其实例没有任何的关联，所以他不能调用得到这个类的其实信息。

class method

类方法刚可以通过cls得到类的其他信息，比如调用其它的类方法。
