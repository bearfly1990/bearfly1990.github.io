三、初始化与访问

3.1 验证服务启动成功

3.2 Web 界面访问与注册

  1. 打开浏览器，输入地址：http://你的Linux机器IP:2283（默认端口2283）

  2. 首次访问进入注册页面，创建管理员账号：

    • 邮箱：任意格式（如 admin@immich.local）

    • 密码：自定义强密码（如 ImmichAdmin@123）

  3. 点击 Sign up 完成注册，自动登录主界面

3.3 防火墙端口放行（若访问不了）

3.4 移动端备份配置（可选）

  1. 手机安装 Immich App：

    • iOS：App Store 搜索“Immich”

    • Android：Google Play 或GitHub 下载APK

  2. 打开 App，输入配置：

    • Server URL：http://你的Linux机器IP:2283

    • 账号/密码：步骤3.2创建的管理员账号

  3. 开启“Auto Backup”，自动同步手机照片/视频

四、常用运维命令

操作 命令
停止 Immich sudo docker compose down
重启 Immich sudo docker compose restart
升级 Immich 到最新版 cd ~/opt/immich && sudo docker compose pull && sudo docker compose up -d
查看所有容器日志 sudo docker compose logs -f
查看存储目录占用空间 du -sh ~/opt/immich/upload
强制重启（容器卡死时） sudo docker compose kill && sudo docker compose rm -f && sudo docker compose up -d
